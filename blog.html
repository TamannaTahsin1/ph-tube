Discuss the scope of var, let, and const

Var: The scope of a var variable is functional scope.It can be declared without initialization.
Let: The scope of a let variable is block scope. It can be declared without initialization.
Const: The scope of a const variable is block scope. It cannot be declared without initialization

Tell us the use cases of null and undefined!

Undefined: Undefined means the variable has been declared, but its value has not been assigned.
Null: Null means an empty value or a blank value. The typeof() operator returns undefined for an undefined variable.
